# Specs Location

The spec documents for this project live at **`~/bandw/specs/`** (outside this repo) because they contain sensitive screenshots that must not be committed.

## IMPORTANT

**NEVER move the spec files back into this repository directory.** They are kept external intentionally to avoid committing sensitive content to version control.

## Implementation Plans

Implementation plans are **not** specs. They do not contain sensitive screenshots and **should** be committed to the repo under `docs/plans/`.

## Reading Specs

To read the specs:

```bash
ls ~/bandw/specs/
cat ~/bandw/specs/<spec-name>.md
```
